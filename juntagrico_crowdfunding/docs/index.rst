Welcome to the juntagrico-crowdfunding documentation
==========================================================
It allows you to create (public) crowdfunding campaigns in juntagrico.

juntagrico-crowdfunding is an extension for juntagrico.

Head over to 
`juntagrico.readthedocs.io <https://juntagrico.readthedocs.io/en/latest/>`_ for information on how to setup juntagrico in order to be able to use this app.
